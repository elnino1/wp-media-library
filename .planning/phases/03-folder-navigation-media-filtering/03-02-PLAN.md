**1. Defensive `buildTree` function:**

```javascript
const buildTree = (folders, parentId = 0) =>
```

With a defensive version:
```javascript
const buildTree = (folders, parentId = 0) => {
    if (!Array.isArray(folders)) return [];
    return folders
        .filter((f) => f && typeof f.id !== 'undefined' && (f.parent || 0) === parentId)
        .map((f) => ({ ...f, children: buildTree(folders, f.id) }));
};
```

This adds:
- `Array.isArray` guard against non-array input
- `f && typeof f.id !== 'undefined'` guard against null/undefined items and items missing `id`

```javascript
const toggleCollapse = (folderId) => {
    setCollapsedIds((prev) => {

            next.add(folderId);

};
```
